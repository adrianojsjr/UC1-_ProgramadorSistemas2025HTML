O que é CSS?

Introdução ao CSS

O CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é uma linguagem de estilo utilizada para definir a aparência visual de páginas web. Ao contrário do HTML, que é responsável pela estrutura de uma página, o CSS lida com o design, o layout e as cores, tornando a experiência do usuário mais agradável e intuitiva.

O que o CSS faz?

O CSS tem várias funcionalidades essenciais para o design de sites. Ele permite a personalização de fontes, cores, margens, espaçamentos, tamanhos, layouts e muitos outros aspectos. Com o uso de CSS, um desenvolvedor web pode:

  • Definir as cores de texto e fundo.
  • Ajustar o espaçamento entre elementos.
  • Escolher fontes e tamanhos de texto adequados.
  • Aplicar efeitos de transição e animações.
  • Controlar a posição de elementos (ex: alinhamento, flutuação, posicionamento fixo).
  • Alterar o layout da página conforme a tela (responsividade).

Como o CSS é utilizado?

O CSS pode ser incluído em um site de três formas principais:

  1. Inline: Através do atributo style diretamente em uma tag HTML.
  2. Interno: Dentro de uma tag <style> no cabeçalho da página HTML.
  3. Externo: Utilizando um arquivo CSS separado e vinculado ao HTML através da tag <link>.

Exemplo de código CSS

Aqui está um exemplo simples de como você pode usar o CSS para estilizar uma página HTML: 

            
    body {
        background-color: #f0f0f0;
        font-family: Arial, sans-serif;
    }
    
    h1 {
        color: #333;
        text-align: center;
    }
    
    p {
        color: #555;
        line-height: 1.6;
    }

Conclusão

O CSS é uma ferramenta fundamental para qualquer desenvolvedor web, permitindo criar sites visualmente atraentes e funcionais. Ele se complementa ao HTML, proporcionando a capacidade de separar o conteúdo da apresentação, o que facilita a manutenção e a escalabilidade de sites.
